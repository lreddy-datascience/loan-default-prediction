# LendingClub Loan Default Prediction

Exploratory data analysis and machine learning to predict borrower default risk on real lending data.

## Overview
This project analyzes the LendingClub loan dataset to identify key trends and risk indicators, then builds classification models to predict which borrowers are likely to default.

## Approach
1. **EDA** — explored loan amounts, interest rates, income, and default rates across borrower segments
2. **Data cleaning** — handled missing values, encoded categorical variables (loan purpose, grade, employment length)
3. **Modeling** — trained classification models to predict default probability
4. **Evaluation** — assessed model performance and identified the strongest risk indicators

## Tech Stack
Python · pandas · scikit-learn · Statistics · Matplotlib

## Key Findings
- Identified key borrower attributes most predictive of default risk
- Built a reusable pipeline for risk scoring on new loan applications
